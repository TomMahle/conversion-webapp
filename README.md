# conversion-webapp

Web application that allows users to convert between units of volume, mass, or temperature. The unit options were chosen based on common cooking units.

## Units Included

### Volume

- Teaspoon
- Tablespoon
- Fluid Ounce
- Cup
- Pint
- Quart
- Gallon
- Liter
- Milliliter

### Mass

- Pounds
- Ounces
- Grams

### Temperature

- Fahrenheit
- Celsius
